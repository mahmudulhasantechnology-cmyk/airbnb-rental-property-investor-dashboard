<h2>📌 Overview</h2>
<p>
This project presents a comprehensive <b>Airbnb Rental Property Investor Dashboard</b> built using <b>Microsoft Excel</b>.  
It analyzes Airbnb listing data to help investors identify <b>profitable locations</b>, <b>pricing strategies</b>, and <b>key factors influencing revenue</b>.
</p>

<p>
🎯 <b>Goal:</b> Enable data-driven decision-making for short-term rental property investments.
</p>

<hr>

<h2>🎯 Objectives</h2>
<ul>
  <li>✔ Identify high-performing locations for Airbnb investments</li>
  <li>✔ Analyze price vs revenue relationships</li>
  <li>✔ Evaluate occupancy and availability trends</li>
  <li>✔ Understand the impact of property features on earnings</li>
  <li>✔ Build an interactive dashboard for investor insights</li>
</ul>
